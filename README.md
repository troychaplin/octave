# Octave: One Theme, Many Keys

Octave is a versatile WordPress block theme built around the idea that a single composition can be played in many keys. Choose from a growing library of hand-tuned style variations — each one a complete shift in color, typography, pattern library, and custom interaction behavior — and watch the whole site change register without rewriting a note.

Powered by Gutenberg's global styles and extended with custom CSS and JavaScript, Octave gives designers and developers a foundation that moves from quiet and editorial to bold and expressive with a single switch.
