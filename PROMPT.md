# Design Prompt: Octave — Default Style Variation

## Project Context

I'm designing **Octave**, a versatile WordPress block theme built around Gutenberg's style variations system. The theme will ship with multiple style variations that swap the entire visual system — colors, typography, patterns, and custom interaction behaviors — while preserving the underlying structure.

I need a design for the **default style variation**: the one that loads first, sets the tone for the brand, and acts as the neutral foundation from which other variations will diverge.

## Design Direction

**Aesthetic:** Clean, crisp, sterile. Think Swiss design principles, modern editorial publications, premium SaaS marketing sites, and architectural portfolios. Generous whitespace. Confident restraint. Nothing decorative for the sake of it.

**Mood words:** Quiet, precise, intentional, considered, editorial, weightless.

**Avoid:** Trendy gradients, glassmorphism, drop shadows, decorative illustrations, playful microinteractions, warm "friendly tech" aesthetics, anything that feels Web 2.0, Web3, or AI-generic.

## Visual System

**Color palette:** Predominantly white or near-white background. One deep neutral for text (consider off-black, deep charcoal, or a desaturated navy rather than pure black). One accent color, used sparingly — ideally a single muted, sophisticated hue. Optional secondary neutral for subtle hierarchy (e.g., light gray dividers or section backgrounds).

**Typography:** Strong typographic hierarchy is the primary design element. Suggest a pairing — likely a refined sans-serif for headings and body, or a sans/serif pairing if it elevates the editorial feel. Generous line-height. Tight, deliberate letter-spacing where appropriate. Display sizes should feel confident, not loud.

**Layout:** Grid-driven. Strong vertical rhythm. Asymmetry is welcome where it serves clarity. Components should breathe — err toward more whitespace than feels comfortable.

**Detail work:** Hairline rules instead of borders. Subtle hover states. Motion, if any, should be quiet — short durations, simple easing, nothing bouncy.

## What I Need Designed

Show me a complete picture of how this style variation looks across the key surfaces of a WordPress site:

1. **Homepage** — hero section, a content section or two, footer
2. **Typography specimen** — display headings, body copy, links, lists, blockquote
3. **Block patterns** — at least three: a hero pattern, a feature/content grid pattern, and a call-to-action pattern
4. **Core blocks styled** — buttons (primary and secondary), form inputs, image with caption, pull quote
5. **Header and footer** — including navigation treatment
6. **A single blog post layout** — to show how the editorial side handles long-form reading

## Constraints to Respect

- This must work as a WordPress block theme, so designs should respect the constraints of Gutenberg's block system (no custom layouts that couldn't be built from blocks and patterns).
- The design needs to be the **starting point** — clean enough that subsequent style variations can build on the same structure with dramatically different palettes, type, and pattern energy without breaking the bones underneath.
- Accessibility matters: contrast ratios, focus states, and readable type sizes should all be considered.

## Deliverable

Static visual designs (any tool — Figma, mockups, image exports). I'm collecting interpretations from several designers and will synthesize the strongest elements into the final direction, so don't hold back on your point of view — the more distinctive your interpretation of "clean and crisp," the more useful it is to me.
